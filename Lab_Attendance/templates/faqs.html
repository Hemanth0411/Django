Frequently Asked Questions

How to add student data?
Method 1: Manually enter student information by filling out the form on the ‘Add Students’ page in the admin dashboard and submit it.

Method 2: Click the ‘Import Students’ option on the ‘Add Student’ page and upload an Excel file with student data. The file should include rows with the following columns:
  • 'roll_no': A 12-digit number
  • 'name'
  • 'year': In the format '2022-26'
  • 'batch': A single uppercase letter, such as 'A' or 'B'
How to clear student data?
There are two methods to clear student data:

Method 1: Navigate to the Student List page. Locate the specific record you wish to delete and click on Delete to remove that record.

Method 2: On the Student List page, use the filters to select a specific year or year and batch. Then, click on Clear Records to delete all student data for the selected criteria.
How to clear session data?
To clear session data, follow these steps:

1. In the Admin Dashboard:
Click on Clear Sessions.

2. Confirm the action:
Select 'Yes' to proceed with clearing the session data.
How to add session data?
There are two methods to add session data:

1. Automatic Addition:
Session data is automatically added whenever attendance is recorded.

2. Manual Upload:
Go to the Session Summary page in the Admin Dashboard and click on the Upload Session button. Then, upload an Excel file containing the session data with the following columns:

  • ID
  • Roll No
  • Name
  • Date
  • Subject
  • Lab
  • In Time
  • Out Time
How to add/modify subjects?
To add or modify subjects, follow these steps:

1. Update the Form:
In forms.py, make the necessary changes in the ReduceAttendanceForm class.

2. Modify the Model:
In models.py, add or modify a variable in the Student class and in the reduce_student function belonging to the Student class.

3. Apply Migrations:
Run the following commands in the shell:
python manage.py makemigrations
python manage.py migrate

4. Update the Views:
In views.py, make appropriate changes in the dictionary within the record_attendance and attendance_summary views.

5. Modify the Template:
In attendance_summary.html, add or modify a column in the table. Update the code in the 'if' block and 'else' block within the <td> tag.
How to add/modify Year?
To make changes to the year, follow these steps:

First: If you want to remove a year for a graduating batch, first delete the student records belonging to that year, and then modify the year itself.

Steps:

1. Update the Forms:
In forms.py, add year options to all instances of the year variable. This includes:
  • StudentForm
  • YearBatchForm
  • SessionFilterForm
  • PartFilter classes

2. Modify the HTML Templates:
In add_students.html, edit_student.html, and student_list.html, update the if-else statements in the <script> section.

3. Update Year Selector:
In student_list.html, also modify the options in the Year selector within the <body> tag.
